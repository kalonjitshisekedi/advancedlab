# Advanced Lab Solutions Website

## Overview
Professional static website for Advanced Lab Solutions, a mineral processing laboratory based in Germiston, South Africa.

## Project Architecture
- **Type:** Static HTML/CSS Website
- **Hosting:** Optimized for GitHub Pages
- **Styling:** Custom CSS (`styles.css`)
- **Images:** Professional industry-specific stock images in `attached_assets/stock_images/`
