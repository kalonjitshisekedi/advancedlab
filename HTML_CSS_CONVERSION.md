## How to Use

### Local Testing
The website is already running! Visit: **http://localhost:5000**

### Deploy to GitHub Pages

1. **Create GitHub Repo**
   ```
   https://github.com/new
   Repository name: advanced-lab-solutions
   Make it: Public
   ```

2. **Push your code**
   ```bash
   git init
   git add .
   git commit -m "Convert to HTML/CSS only"
   git remote add origin https://github.com/USERNAME/advanced-lab-solutions.git
   git push -u origin main
   ```

3. **Enable GitHub Pages**
   - Go to repo Settings ‚Üí Pages
   - Source: Deploy from branch
   - Branch: main
   - Folder: / (root)
   - Click Save

4. **Your site is live**
   - At: `https://USERNAME.github.io/advanced-lab-solutions/`

### Add Custom Domain (GoDaddy)

1. **Point DNS to GitHub**
   - Go to GoDaddy DNS Management
   - Add A records:
     ```
     185.199.108.153
     185.199.109.153
     185.199.110.153
     185.199.111.153
     ```
   - Add CNAME for www:
     ```
     www ‚Üí USERNAME.github.io
     ```

2. **Configure in GitHub**
   - Settings ‚Üí Pages
   - Custom domain: yourdomain.co.za
   - Save (wait ~5 min for HTTPS)

3. **Done!** Site accessible at your custom domain
